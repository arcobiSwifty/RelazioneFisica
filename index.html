Obbiettivo:

Calcolare il coefficeinte di attrito statico

Materiali e Strumenti:

    -Un cubo di legno con una massa di 140 grammi (errore assoluto: 1g) con un cordino e due lati ricoperti da velcro
    -Un cubo di legno con una massa di 140 grammi (errore assoluto: 1g)
    -Un dinamometro con portata di un 1 Newton e sensibilita' di 0,01 N
    -Due cilindri di legno dalla massa di 36 e 38 grammi (errore assoluto di ciascuno: 1g)


Procedimento:

    Per misurare la forza di primo distacco, prendere la il cubo di legno con il cordino e posizionare sopra di esso la dovuta massa. Successivamente, agganciare la parte mobile del dinamometro all'estremita' libera del cordino di legno, per poi iniziare a applicare una forza man mano crescente con la mano.
    Appena il cubo di legno comincia a muoversi, smettere di applicare una forza con la mano e registrare l'intensita' della forza in newton riportato sul dinamometro.
    Ripetere questo esperimento per ogni combinazioni possibili (mantenendo il cubo di legno come base: un cubo, un cubo e un cilindro, un cubo e due cilindri, due cubi, due cubi e un cilindro, due cubi e due cilindri), otto volte.

    Da questi dati, per calcolare il coefficiente di attrito statico basta applicare la formula inversa di:

    Attrito statico massimo = Forza premente x Coefficiente di attrito statico

    Coefficiente di attrito statico = Attrito statico massimo / Forza premente

    Calcolare ora la media delle forze di primo distacco per ottenere l'attrito statico massimo per ogni serie:

    (0,35 N + 0,35 N + 0,33 N +0,37 N + 0,39 N + 0,37 N + 0,38 N + 0,37 N) / 8 = 0,36 N

    (0,50 N + 0,46 N + 0,48 N + 0,44 N + 0,50 N + 0,50 N + 0,46 N + 0,45 N) / 8 = 0,47 N

    (0,54 N + 0,60 N + 0,60 N + 0,54 N + 0,56 N + 0,54 N + 0,55 N + 0,62 N) / 8 = 0,57 N

    (0,72 N + 0,65 N + 0,73 N + 0,70 N + 0,69 N + 0,69 N + 0,72 N + 0,70 N) / 8 = 0,70 N

    (0,75 N + 0,76 N + 0,75 N + 0,74 N + 0,78 N + 0,73 N + 0,73 N + 0,73 N) / 8 = 0,75 N

    (0,81 N + 0,85 N + 0,84 N + 0,85 N + 0,80 N + 0,86 N + 0,86 N + 0,87 N) / 8 = 0,84 N

    Calcolare la forza premente di ogni serie di misurazioni:

    0,140 kg x 9,80 m/s = 1,37 N

    0,176 kg x 9,80 m/s = 1,72 N

    0,214 kg x 9,80 m/s = 2,10 N

    0,280 kg x 9,80 m/s = 2,74 N

    0,316 kg x 9,80 m/s = 3,10 N

    0,354 kg x 9,80 m/s = 3,47 N

    Con i dati appena ottenuti, calcolare il valore assoluto dei coefficienti di attrito statico e successivamente farne la media:

    0,36 N / 1,37 N = 0,26

    0,47 N / 1,72 N = 0,27

    0,57 N / 2,10 N = 0,27

    0,70 N / 2,74 N = 0,26

    0,75 N / 3,10 N = 0,24

    0,84 N / 3,47 N = 0,24

    (0,26 + 0,27 + 0,27 + 0,26 + 0,24 + 0,24) / 6 = 0,26

    Calcolare l'errore assoluto del coefficiente di attrito statico come la semidispersione di tutti i valori ottenuti precedentemente:

    (0,27-0,24)/2 = 0,015

    Il coefficiente di attrito statico fra velcro e legno (levigato) e' 0,26 con errore assoluto di 0,015.

Conclusioni:

    Possiamo osservare che il coefficiente di attrito statico non varia a seconda a seconda del peso del corpo, in quanto e' il rapporto fra due misure estensive.
